# ORFE Senior Thesis Code

Code for my Princeton ORFE senior thesis, “Predicting Shareholder Activism Risk: A Machine-Learning-Based Approach for Corporate Defense Strategy.”

This repository contains the scripts used to run the target-prediction and outcome-prediction model horse races. The target prediction interpretability stage is in the target runs folder.

Raw data and the two final merged modeling panels (for target- and outcome-prediction) are not included in this repository because they were constructed using restricted-access sources, including WRDS, ISS, BoardEx, and LSEG.

Main folders:
- `code/target_runs/`
- `code/outcome_runs/`
